# Shared Modules

This directory stores reusable frontend modules used by the current multi-page app.

## Current contents

- API helpers for species, QA, records, and geocoding
- Reusable map utilities
- Feature panels for the case pages
- Composables that hold shared interaction logic

## Used by

- `src/features/CaseSpeciesFeature.vue`
- `src/features/CaseQaFeature.vue`
- `src/features/CaseReportFeature.vue`
- `src/legacy/LegacyCaseFeaturePage.vue`
