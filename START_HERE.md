## ⏱️ 3 步快速启动（5 分钟）

### 第 1 步：安装数据库依赖（30 秒）

```bash
cd backend
pip install -r requirements.txt
```

### 第 2 步：迁移 CSV 数据到数据库（2 分钟）

```bash
python migrate_csv_to_db.py
```

你会看到：
```
✓ 数据库初始化完成
✓ 发现 8 个物种 CSV 文件
✓ 导入 13,207 条记录
✅ 迁移完成！
```

### 第 3 步：启动应用（10 秒）

```bash
# Windows 用户：双击
start.bat

# 或手动启动：
cd backend && uvicorn main:app --reload --port 8000
cd frontend && npm run dev
```

**然后你会看到：**
- ✅ 浏览器自动打开 http://localhost:5173
- ✅ 水生入侵物种平台框页面加载
- ✅ 物种列表显示 8 个物种
